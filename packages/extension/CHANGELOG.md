# Arela VS Code Extension Changelog

All notable changes to the Arela VS Code extension will be documented in this file. The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [5.0.0] - 2024-07-01
### Added
- Fully redesigned ChatOps experience with CTO-grade prompts and runway visibility.
- First-class Ollama support for local inference, complete with model switcher.
- Visual assets (icon, banner screenshots, hero GIF) for the Marketplace listing.
- Automated publishing workflow that deploys on GitHub release creation.

### Changed
- Updated README and marketplace metadata to highlight multi-surface AI co-founder workflows.

## [4.3.0] - 2024-06-08
### Added
- Multi-model routing (OpenAI + Anthropic) and context-aware conversation resets.
- Workspace context upload command with streaming progress events.

### Fixed
- Windows shell path handling for `Add Workspace Context` command.
- Various UI focus issues inside the VS Code webview.

## [4.2.0] - 2024-05-15
### Added
- Draft PR review summaries directly inside the Arela side panel.
- Smoother onboarding flow with provider-specific quick start tips.

### Fixed
- Debounced log spam when activation happens during VS Code startup.
- Minor accessibility fixes for screen reader announcements.

## [4.1.0] - 2024-04-01
### Added
- Initial public preview of the Arela VS Code extension with chat panel and context ingestion commands.
