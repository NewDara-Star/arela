# TEST-001: Create a Simple Function

## Objective
Create a simple TypeScript function that adds two numbers.

## Requirements
- Function name: `add`
- Takes two parameters: `a: number` and `b: number`
- Returns the sum as a number
- Include a JSDoc comment

## Acceptance Criteria
- [ ] Function is properly typed
- [ ] JSDoc comment explains what it does
- [ ] Returns correct sum

## Example
```typescript
add(2, 3) // returns 5
```

## Output Format
Please provide the complete function code.
