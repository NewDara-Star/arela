# TEST-002: Create a Multiply Function

## Objective
Create a simple TypeScript function that multiplies two numbers.

## Requirements
- Function name: `multiply`
- Takes two parameters: `a: number` and `b: number`
- Returns the product as a number
- Include a JSDoc comment

## Acceptance Criteria
- [ ] Function is properly typed
- [ ] JSDoc comment explains what it does
- [ ] Returns correct product

## Example
```typescript
multiply(3, 4) // returns 12
```

## Output Format
Please provide the complete function code.
